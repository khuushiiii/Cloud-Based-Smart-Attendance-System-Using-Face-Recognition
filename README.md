Cloud-Based Smart Attendance System

A smart attendance management system that uses Face Recognition and Cloud Technology to automatically detect and mark attendance in real time. The project reduces manual work, prevents proxy attendance, and stores attendance records securely in the cloud.

Features
Real-time face recognition
Automatic attendance marking
Secure cloud database integration
Web dashboard for monitoring attendance
Reduces human errors and fake attendance
Fast and contactless system

Technologies Used
Python
DeepFace
OpenCV
Firebase
HTML, CSS, JavaScript
Flask

How It Works
Capture face using webcam/camera
Detect and recognize faces using DeepFace
Match face with stored database
Mark attendance automatically
Store records in Firebase Cloud Database
Display attendance on web dashboard
